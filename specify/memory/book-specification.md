# Book Specification

Title: Designing Your Life with AI
Subtitle: Practical workflows, prompts and projects to design a meaningful life with AI support
Length target: 25,000 to 35,000 words
Structure: 10 chapters, each 2,000–4,000 words, plus appendices with prompts and templates.
Audience: Readers who want to use AI to clarify values, design experiments, and ship small projects that improve life quality.
Delivery formats:
- PDF book
- Docusaurus documentation site
- A prompts package in specify/memory/prompts
Milestones:
1. Outline and Chapter 1 complete
2. Draft chapters 2–5
3. Draft chapters 6–10
4. Edit and compile PDF
5. Build Docusaurus site and publish
Core assets:
- docs/chapters/*.md
- specify/memory/prompts/*.md
- src/styles for site
Startup tasks:
- Populate chapter-1-intro.md with full intro
- Create prompts folder and add 20 starter prompts
- Add README with quickstart
