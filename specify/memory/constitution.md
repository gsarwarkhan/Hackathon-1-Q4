# Book Constitution

Project: Designing Your Life with AI
Owner: [Your Name]
Purpose: Practical guide that helps readers design a meaningful life using AI tools and workflows.
Target audience: Students, professionals, freelancers, and early adopters who want pragmatic AI workflows for life design and career decisions.
Deliverables:
- A short book (approx 25,000–35,000 words)
- Downloadable PDF
- Docusaurus site with chapter pages and code examples
- A companion folder with prompts and templates (specify/memory/prompts)
Guiding principles:
1. Practicality: real steps users can copy and run.
2. Privacy: keep personal or sensitive data local and secure.
3. Reproducibility: include prompts, commands and assets so readers can reproduce outcomes.
4. Clean structure: chapters follow a learn->apply->exercise pattern.
Scope:
- Core topics: self-assessment, career design, habit design, decision frameworks, AI tools, prompt recipes, project templates.
- Exclusions: deep AI research theory and raw model internals.
Maintenance:
- Use Git for versions.
- Keep all working files in this repo.
